## Start

Для запуска проекта локально, необходимо его скачать

`git clone https://github.com/DmitriyShvanyk/toxin.git`


## Development

Для запуска dev версии

`npm run dev`


## Production

Для запуска production версии

`npm run build`



## Deploy

Для запуска deploy версии

`npm run deploy`
